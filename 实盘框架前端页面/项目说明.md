# 实盘交易管理系统前端 - 项目说明

## 📖 项目概述

这是一个专为实盘交易框架开发的现代化Web前端系统，用于管理和监控：
- **C++实盘交易框架**：负责订单执行、行情接收等底层功能
- **Python交易策略**：多种量化交易策略的部署和运行
- **多OKX账户**：支持管理多个OKX交易账户

## ✨ 核心特性

### 1. 🎯 策略管理
- **全生命周期管理**：创建、启动、停止、删除策略
- **实时监控**：查看策略运行状态、盈亏、收益率
- **多策略分类**：
  - 网格交易策略
  - 趋势跟踪策略
  - 套利策略
  - 做市策略
  - 自定义策略
- **Python集成**：通过界面配置Python策略文件路径和参数

### 2. 💰 账户管理
- **多账户支持**：可添加多个OKX账户
- **实时数据同步**：余额、净值、持仓实时更新
- **安全配置**：API Key加密存储，支持模拟盘/实盘切换
- **净值追踪**：
  - 净值曲线图（1天/7天/30天/全部）
  - 收益率统计
  - 最大回撤分析

### 3. 📊 订单管理
- **全方位查询**：
  - 按交易对筛选
  - 按状态筛选（已创建/已提交/已成交/已取消等）
  - 按订单类型筛选（限价/市价）
  - 按方向筛选（买入/卖出）
  - 时间范围筛选
- **手动下单**：支持手动下限价单和市价单
- **批量操作**：批量取消订单
- **订单详情**：查看成交记录、手续费等详细信息
- **数据导出**：导出订单数据用于分析

### 4. 📈 持仓管理
- **持仓总览**：
  - 持仓市值统计
  - 未实现盈亏
  - 盈利/亏损持仓数量
- **可视化展示**：
  - 持仓分布饼图
  - 盈亏分布柱状图
- **风险监控**：
  - 杠杆倍数
  - 强平价格预警
- **快速操作**：一键平仓

### 5. 📉 数据可视化
- **实时图表**：基于ECharts的高性能图表
- **净值曲线**：清晰展示账户净值变化趋势
- **策略性能对比**：多策略收益率排行
- **持仓分析**：资产分布和盈亏情况


## 🔌 后端对接

### 接口规范

所有接口遵循RESTful设计，返回格式：

```json
{
  "code": 200,
  "message": "success",
  "data": {...}
}
```

### 主要接口

#### 策略接口
- `GET /api/strategies` - 获取策略列表
- `POST /api/strategies/:id/start` - 启动策略
- `POST /api/strategies/:id/stop` - 停止策略
- `POST /api/strategies` - 创建策略
- `DELETE /api/strategies/:id` - 删除策略

#### 账户接口
- `GET /api/accounts` - 获取账户列表
- `GET /api/accounts/:id/balance` - 获取余额
- `GET /api/accounts/:id/equity-curve` - 获取净值曲线
- `POST /api/accounts` - 添加账户
- `POST /api/accounts/:id/sync` - 同步账户

#### 订单接口
- `GET /api/orders` - 获取订单列表
- `POST /api/orders` - 手动下单
- `POST /api/orders/:id/cancel` - 取消订单
- `POST /api/orders/batch-cancel` - 批量取消
- `GET /api/positions` - 获取持仓列表

### Mock数据模式

系统内置Mock数据服务，可独立运行进行开发和测试。

切换到真实后端：
1. 编辑 `src/api/mock.js`
2. 设置 `USE_MOCK = false`
3. 配置 `vite.config.js` 中的后端地址
