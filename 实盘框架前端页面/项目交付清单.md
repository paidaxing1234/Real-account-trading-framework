## 🔌 后端接口要求

系统需要后端提供以下RESTful接口：

### 策略接口
- `GET /api/strategies` - 获取策略列表
- `GET /api/strategies/:id` - 获取策略详情
- `POST /api/strategies/:id/start` - 启动策略
- `POST /api/strategies/:id/stop` - 停止策略
- `POST /api/strategies` - 创建策略
- `DELETE /api/strategies/:id` - 删除策略

### 账户接口
- `GET /api/accounts` - 获取账户列表
- `GET /api/accounts/:id/balance` - 获取账户余额
- `GET /api/accounts/:id/equity-curve` - 获取净值曲线
- `POST /api/accounts` - 添加账户
- `POST /api/accounts/:id/sync` - 同步账户

### 订单接口
- `GET /api/orders` - 获取订单列表
- `POST /api/orders` - 下单
- `POST /api/orders/:id/cancel` - 取消订单
- `GET /api/positions` - 获取持仓

**注意**: 当前使用Mock数据，可独立运行。

## ⚙️ 配置说明

### 切换到真实后端
编辑 `src/api/mock.js`:
```javascript
const USE_MOCK = false  // 改为 false
```

### 修改后端地址
编辑 `vite.config.js`:
```javascript
proxy: {
  '/api': {
    target: 'http://localhost:8000',  // 修改此处
    changeOrigin: true
  }
}
```

## 📞 技术支持

### 问题排查
1. 检查Node.js版本 >= 16
2. 清理并重新安装依赖: `rm -rf node_modules && npm install`
3. 查看浏览器控制台错误
4. 检查后端服务状态

### 常见问题
- **页面空白**: 检查路由配置和组件导入
- **数据不显示**: 检查API地址和Mock配置
- **样式错乱**: 清除浏览器缓存
